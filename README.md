# My eCommerce App

## Summary

A web based e-commerce app to discover new and trending games. Features include wishlist to bookmark games, video games list with search, sort and filter capability, video game detail to see the detail of each game. Here are the details for this app:

### Tech Stack

* Next.js
* MongoDB
* Rest API
* Vercel
